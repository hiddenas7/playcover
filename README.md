<h3 id="h3--"><a name="💡 原神无法使用手机验证码登录" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 原神无法使用手机验证码登录</h3><p>出现智能性分析等，我们可以完全退出 playCover 后重启，如果不行在原神图标右键清除应用数据后重启<br>以上方法都不行，就只能卸载原神后重新安装，进入游戏登录页面后，切记乱点，忘记密码等的都不要点（容易出问题），直接手机验证码登录就好</p>
<h3 id="h3--"><a name="💡 原神使用密码登录" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 原神使用密码登录</h3><p>建议使用手机短信验证码登录，如果必须要使用密码登录，则需要关闭 mac 电脑 SIP，sip关闭麻烦不说，就怕搞不好电脑出问题（谨慎设置），SIP关闭请自行百度</p>
<h3 id="h3--"><a name="💡 键盘映射或把鼠标移动没有反应" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 键盘映射或把鼠标移动没有反应</h3><p>playCover 存在两种控制方式 一是鼠标指针控制 二是键盘映射控制，二者不可同时生效需要使用 <code>⌥ option</code> 进行来回切换,出现问题后我们可以来回切换控制状态来恢复控制，如果还不行就退出重新进去游戏</p>
<h3 id="h3--"><a name="💡 键盘映射配置后不生效或部分按键失灵" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 键盘映射配置后不生效或部分按键失灵</h3><p>playCover 的键盘映射是不能存在冲突按键的，就是同一个按键只能匹配同一个按钮，尤其注意移动的wasd按键，因为移动默认是wasd，小伙伴可能没注意到又设置了别的按钮，导致出现问题</p>
<h3 id="h3--"><a name="💡 金铲铲备战席被遮挡等图形界面问题" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 金铲铲备战席被遮挡等图形界面问题</h3><p>playCover 默认为手机大小比例，但某些游戏可能使用的 ipad 布局导致出现问题，我们解决无非就两种思路，一种是在游戏内设置，让游戏成为手机布局，另一种<code>推荐</code>是进游戏前，先在游戏图标右键 - “应用偏好设置” - 打开窗口大小调整并下下方选择一个 ipad 的设备等</p>
<h3 id="h3--"><a name="💡 授权金铲铲麦克风权限" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 授权金铲铲麦克风权限</h3><p>进入 系统偏好设置 - 选择安全性与隐私 - 选择麦克风 - 找到金铲铲游戏（jkchess）选中即可</p>
<h3 id="h3--"><a name="💡 金铲铲闪退、金铲铲没有声音（已授权麦克风权限）" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 金铲铲闪退、金铲铲没有声音（已授权麦克风权限）</h3><p>1.右键点击金铲铲图标 -&gt; 点击“在访达中查看”<br><img src="https://www.hereitis.cn/profile/editor/2022/12/13/7575540c75be4486bfe17ee284bf02b0.png" alt=""></p>
<p>2.重命名“金铲铲之战”包文件为“jkchess”<br><img src="https://www.hereitis.cn/profile/editor/2022/12/13/c723c6bb01b4471da6323674605aaadc.png" alt=""><br>3.双击包文件直接进入游戏、或重启 playCover 后进去该游戏</p>
<h3 id="h3--"><a name="💡 任意游戏进去只有图标无界面，有声音无画面或出现卡界面，一直意外退出等情况" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 任意游戏进去只有图标无界面，有声音无画面或出现卡界面，一直意外退出等情况</h3><p>出现这些情况可能是分辨率不对，可以游戏图标右键-&gt;点击设置</p>
<p>调整为一个合适的分辨率就可以了<br><img src="https://www.hereitis.cn/profile/editor/2023/04/27/fba0e277b33a4e5f8fc7947253f8bf81.png" alt=""></p>
<h3 id="h3--"><a name="💡 崩坏：星穹铁道无声音、无按钮、卡界面" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 崩坏：星穹铁道无声音、无按钮、卡界面</h3><p>出现无声音，无按钮等问题的小伙伴们，游戏图标右键-&gt;在访达中查看-&gt;然后将游戏名称改为 bhxqtd 之后，双击打开就可以了，改名之后在playCover里第一次打开该游戏会失败，第二次就好了</p>
<h3 id="h3--macos-14-"><a name="💡 升级 MacOS 14 后游戏打不开、按键失效" class="reference-link"></a><span class="header-link octicon octicon-link"></span>💡 升级 MacOS 14 后游戏打不开、按键失效</h3><p>升级到 MacOS 14 以后部分游戏可能会打不开，或者出现按键失效等情况，我们需要下载升级 playCover 到最新的测试版 playCover nightly 446 以上</p>
